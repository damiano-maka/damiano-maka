## 🚀 About Me

I am a **Frontend Engineer** who bridges the gap between pixel-perfect user interfaces and cutting-edge AI integration. With a strong foundation in modern frontend architectures and a deep understanding of large language models (LLMs), I build applications that are not just responsive, but intelligent.

> **The AI-Accelerated Advantage:** I don't just write code; I leverage advanced prompt engineering and local/cloud LLM orchestration (Claude, Gemini, OpenAI) to build features faster, optimize performance, and integrate smart capabilities directly into the client side.

### 🤖 AI, LLMs & Prompt Engineering
* **Model Orchestration:** Advanced context engineering and system prompting across **Claude 3.5**, **GPT-4o**, and **Gemini 1.5 Pro**.
* **Local AI Deployments:** Running and quantizing open-source models (Llama 3, Mistral) locally using **Ollama** and **LM Studio** for secure, low-latency applications.
* **Under the Hood:** Solid grasp of tokenization, context windows, temperature manipulation, and embedding vectors.
